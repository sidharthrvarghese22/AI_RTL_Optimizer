# AI RTL Optimizer

AI-Assisted RTL Analysis and Optimization Tool using Python and Verilog HDL.

---

## Features

- RTL Parsing
- Timing Analysis
- Optimization Suggestions
- ML Delay Prediction
- RTL Complexity Scoring
- RTL Quality Grading
- Streamlit GUI Dashboard
- Downloadable Analysis Reports


---

## Technologies Used

- Python
- Verilog HDL
- Streamlit
- Pandas
- Scikit-learn
- Regex
- Icarus Verilog
